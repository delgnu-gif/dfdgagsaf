# 🎯 기발한 교실 자리 뽑기 (Creative Class Seat Picker)

> **선생님과 학생 모두가 즐거운 교실을 만드는 창의적인 자리 배치 웹 애플리케이션**
> 
> 지루한 자리 바꾸기는 가라! 다양한 애니메이션 효과와 게임 요소를 도입하여 자리 바꾸는 시간 자체를 하나의 학급 이벤트로 만들어주는 웹앱입니다. 별도의 설치 없이 브라우저에서 바로 실행됩니다.

---

## 🚀 주요 기능 (Key Features)

### 1. 학생 명렬 관리 (Student Management)
* **간편 등록:** 텍스트 줄바꿈(Enter) 분리 또는 엑셀(CSV) 복사·붙여넣기로 학생 명단을 한 번에 등록합니다.
* **데이터 저장:** 브라우저의 `LocalStorage`를 활용하여 새로고침하거나 재방문해도 학생 명단이 유지됩니다.
* **특이사항 설정 (옵션):** 시력 제한(앞자리 고정), 키 고려, 절대 같은 자리에 앉으면 안 되는 '금지 짝꿍' 설정 기능을 지원합니다.

### 2. 교실 구조 설정 (Classroom Layout)
* **그리드 커스텀:** `N x M` 형태로 교실의 행과 열 개수를 자유롭게 조절합니다.
* **장애물 및 빈자리 설정:** 교탁 위치 설정, 사물함이나 통로로 비워두어야 하는 칸을 클릭 한 번으로 비활성화할 수 있습니다.

### 3. 🎨 창의적인 뽑기 모드 (Creative Drawing Modes)

| 모드 이름 | 연출 방식 | 기대 효과 |
| :--- | :--- | :--- |
| **🎰 룰렛 돌리기** | 특정 자리를 먼저 선택한 후, 룰렛이 돌며 학생 이름이 매칭됩니다. | 한 명 한 명 결정될 때마다 극적인 긴장감 연출 |
| **📦 시크릿 상자 뽑기** | 교실 자리 배치도 위에 상자들이 놓여 있고, 학생이 나와서 상자를 클릭하면 이름이 공개됩니다. | 학생들이 직접 참여하는 인터랙티브한 재미 |
| **⚡ 초고속 일괄 배치** | 1초 만에 전체 자리가 화려한 파티클 이펙트와 함께 한 번에 배치됩니다. | 바쁜 학기 중 빠르게 자리를 바꿔야 할 때 유용 |
| **💣 폭탄 돌리기** | 타이머가 가며 폭탄이 터지는 순간 화면에 떠 있는 학생이 해당 자리에 당첨됩니다. | 게임 같은 스릴 제공 |

### 4. 결과 저장 및 공유 (Export & Share)
* **이미지 저장:** 배치 완료된 교실 모습을 `PNG` 이미지 파일로 다운로드합니다.
* **인쇄 최적화:** A4 용지에 깔끔하게 출력할 수 있는 프린트용 CSS 스타일을 지원합니다.

---

## 🛠 기술 스택 추천 (Recommended Tech Stack)

* **Frontend:** HTML5, CSS3 (Tailwind CSS 추천 - 빠른 UI 구현), JavaScript (Vanilla JS 또는 React/Vue.js)
* **Animation:** Canvas API, GSAP (애니메이션 효과 극대화)
* **Libraries:** `html2canvas` (화면 이미지 캡처용), `confetti-canvas` (축하 효과)
